Engineered and deployed a fully automated, end-to-end retail data platform on Azure using Azure Data Factory (ADF). The solution ingests, processes, and serves data from multiple enterprise-grade sources into a unified, analytics-ready model.

## 🚀 Overview

- Multi-source ingestion: CSV, On-prem SQL, REST APIs  
- Domain: Customers, Products, Sales  
- Fully automated with ADF scheduled triggers  

## 🏗️ Architecture

Medallion Architecture (Bronze → Silver → Gold)

## 🧱 Data Layers

**Bronze:** Raw ingestion  
**Silver:** Cleaned & standardized  
**Gold:** Denormalised fact table  

## ⚙️ Orchestration

- Azure Data Factory pipelines  
- Scheduled automation  
- Reusable & fault-tolerant design  

## 🛠️ Tech Stack

- Azure Data Factory  
- ADLS Gen2  
- Azure SQL Database  
- Power BI  

## 🏁 Outcome

Production-style pipeline delivering analytics-ready data for reporting.
